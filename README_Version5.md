2) ادفع التغييرات إلى GitHub.

ملاحظات:
- لا تترك مسارات تبدأ بـ "/" لأن ذلك يشير إلى جذر النطاق؛ الملفات هنا تستخدم مسارات نسبية مع <base href="/rosi/">.
- إذا عندك build من Angular (مجلد dist أو ملفات main.js...) ضع محتواها في نفس المجلد وسيحاول المتصفح تحميلها تلقائياً.
